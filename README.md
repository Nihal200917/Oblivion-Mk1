# Project Oblivion
Build a combat ready 2 wheeled Battle Robot weighing in at 1lb controlled externally by a person and WIN!!!!

## Description
1lb (454g) Antweight Battle Robot
Weapon Type: Flipper

### Goals
- Highly Durable Chassis (recommended to be made with TPU)
- Highly Armored
- Fast and High Torque flipping
- Easy Repairs
- Very Structured Mounts
- Shock Absorbent Body
- High Traction Drivetrain
